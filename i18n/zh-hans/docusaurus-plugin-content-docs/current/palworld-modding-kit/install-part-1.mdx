# Prerequisites - Part 1

## .NET 6
import { FaDownload } from "react-icons/fa6";

For Unreal Build Tools to function correctly, you will need to have .NET 6 installed on your system.

[Download <FaDownload />](https://dotnet.microsoft.com/en-us/download/dotnet/6.0/runtime?cid=getdotnetcore&os=windows&arch=x64)

Click on the `Download x64` button to download the runtime:

![Dotnet download](./assets/Dotnet.png)

Once you've downloaded the installer, open it and install the runtime.

## Visual Studio 2022

To compile the sdk you will need Visual Studio 2022 installed.

[Download <FaDownload />](https://visualstudio.microsoft.com/vs/) 

A community edition version will suffice. While installing it, make sure to check the following components in the installer:


![Visual Studio Installer Options](./assets/VisualStudioInstaller.png)
![Visual Studio Installer Options 2](./assets/VisualStudioInstaller2.png)

Then press continue, and wait for visual studio to install.

import { FaCirclePlus } from "react-icons/fa6";

## Unreal Engine 5.1

After installing Visual Studio, we need to install Unreal Engine.

1) Open Epic Games Launcher
2) Head over to `Unreal Engine` tab
3) Go into `Library` tab
4) Press the <Highlight color="#dea309">Gold <FaCirclePlus /></Highlight> to add a new version

![Add new Unreal Engine version](./assets/EpicUnrealVersion.png)

5) You should see a new card appear, with the ability to select an engine version.
6) Select version `5.1`

![5.1 Version](./assets/Versions.png)

:::warning
Any 5.1 version will work, it doesn't matter if it's 5.1 or 5.1.1
:::

## Wwise

Palworld uses [Wwise](https://www.audiokinetic.com/en/products/wwise/) for its sounds, so we will need to install this too.

:::danger
Even if you are not planning on making sounds, you still need wwise, otherwise the project will not compile
:::

1) To install [Wwise](https://www.audiokinetic.com/en/products/wwise/) click `Download Audikinetic Launcher` [Download <FaDownload />](https://www.audiokinetic.com/en/download)


![Download Audiokinetic Launcher](./assets/WwiseWebsiteDownload2.png)

:::note
This will require you to make an account.
:::

2) After downloading the launcher, install it and login.
2) Head over to the `Wwise` tab.
3) On the bottom left of your screen you should see an `Install A New Version` Header.
4) Select `2021.1.11` as your [Wwise](https://www.audiokinetic.com/en/products/wwise/) version and press install

![Install A New Version](./assets/WwiseInstallNewVersion.png)

:::info
On the next page, select:

- SDK (C++)
- Microsoft -> Windows -> Visual Studio 2022
:::

The selection should look like so:

![Install Options](./assets/WwiseInstallSelection.png)

5) Now press `Next`, On the plugins page you don't need any, so hit `Install`!
6) After installing the [Wwise](https://www.audiokinetic.com/en/products/wwise/) SDK, Head over to the `Unreal Engine` Tab.
7) Press the <FaDownload /> button, and select `Offline Integration Files`

![Offline integration files](./assets/WwiseOfflineIntegrationFiles.png)

8) In the `Integration Version` Choose `2021.1.11` for the Wwise version. Set the directory to a memorable location, and hit install!

### Finished

:::note
You will need this directory later, remember where you installed these files.
:::
